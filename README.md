# Decidim::AdminStats

Display stats on admin side.

## Usage

AdminStats will be available as a Component for a Participatory
Space.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'decidim-admin_stats
```

And then execute:

```bash
bundle
```
